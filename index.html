Toolbox Opened

 

DevTools is transitioning to the new metric DEVTOOLS_TOOLBOX_OPENED_COUNT, which records the number of times a user opens the toolbox per day. Previously DevTools was using the metric DEVTOOLS_TOOLBOX_OPENED_BOOLEAN which only captured the toolbox being opened once per user, per day. As the rollout of the new metric continues this graph will shift over time and have certain releases missing. Also note that the new metric has been changed to opt-out for release so those numbers will also shift as it rolls out into the full release population.

 

The DEVTOOLS_TOOLBOX_OPENED_PER_USER_FLAG will become opt-out for release when bug 1250171 is fixed, however until that time the release (not pre-release) numbers will remain highly inaccurate.

 

Toolbox Time Spent

Over a rolling 6 months this chart shows the amount of time spent (y axis) with the DevTools toolbox opened and the release. Size of the circles indicates the number of users reporting that time slot.
